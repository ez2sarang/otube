## Phase 1 / 초기 7문

**Q1. 프로젝트 목적**
AI 오프라인 분석 파이프라인, 영상(특히 구글) 분석도구

**Q2. 사용자/고객**
외부 사용자 포함. 향후 유료계획 있어, 특정 기능은 제한 예정

**Q3. 배포 환경**
향후 Vercel이나 클라우드 환경 고려

**Q4. 우선순위**
동영상 분석 및 분석내용으로 AI 질의응답

**Q5. AI 제한 사항**
지금은 딱히 없음, 필요하면 요청

**Q6. 성공 기준**
현재 기본 기능 작동 검증됨. 실제 서비스 오픈 + 수익화. 사람들이 많이 찾는 서비스.

**Q7. 빌드/테스트**
지금은 없음. LLM 개발 시 테스트 검증 과정만 거침.

## Phase 2 / Follow-up (4라운드)

**Round 1 — 목적 명확화**
- 세부: VTT 자막 수집 → 없으면 Whisper STT → 슬라이드 이미지 추출 → LLM gateway Q&A
- 타겟: 긴 영상 분석, 문서화, 2차 AI 가공이 필요한 모든 사용자

**Round 2 — LLM 구조 + 다음 작업**
- LLM: paperCompany 프로젝트 서비스, localhost:3100/PIN/llm-gateway
- 다음 작업: 유료결제, 회원관리, 권한별 기능 제한

**Round 3 — 배포 + 결제 스택**
- 배포: 로컬 public IP화 또는 API 비용 지불
- 결제: Stripe + Supabase Auth (1순위), cafe24결제 (대안)

**Round 4 — 서비스 범위 + 검증**
- 언어: 한국어 + 영어 멀티
- 유료 기능: AI 질의응답 권수 제한
- 검증: 현재 직접 확인. 자동화 테스트 도입 필요.
