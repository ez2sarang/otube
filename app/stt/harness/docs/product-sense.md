# STT 앱 프로덕트 센스

## 이 앱이 해결하는 문제

유튜브 영상이나 오디오 파일의 내용을 텍스트로 빠르게 추출하고,
읽기 좋은 문서로 정리하고 싶은 사람들을 위한 도구.

## 핵심 사용자

- AI/개발 관련 유튜브 영상을 자주 보는 사람
- 강의/세미나 내용을 텍스트로 아카이빙하고 싶은 사람
- 한국어 콘텐츠를 주로 다루는 사람

## 핵심 경험

1. URL 하나만 주면 텍스트가 나온다 (원클릭)
2. 한국어 인식 품질이 충분히 좋다
3. 타임스탬프가 있어서 원본 영상과 대조 가능
4. 결과물이 마크다운이라 바로 활용 가능

## 이 앱이 하지 않는 것

- 실시간 음성 인식 (오프라인 배치 처리만)
- 번역 (원본 언어 그대로 출력)
- 화자 분리 (향후 확장 가능)
