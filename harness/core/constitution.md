# 헌법 (Constitution)

이 저장소에서 일하는 모든 AI 에이전트가 따라야 할 규칙.

## 코어 가치

1. **간결함 우선:** 코드와 문서 모두 짧고 읽기 쉽게. 한 파일 500줄 이하.
2. **로컬 퍼스트:** 외부 API 의존 최소화. 오프라인에서도 동작하는 것이 기본.
3. **무료/오픈소스 우선:** 유료 API보다 오픈소스 대안을 먼저 검토.
4. **Apple Silicon 최적화:** Mac M시리즈에서 최적 성능을 내는 도구를 우선 선택.

## Do and Don't

### Do
- 새 의존성 추가 시 반드시 requirements.txt에 고정 버전으로 기록
- venv 안에서만 pip 패키지 설치
- 모든 오디오/비디오 처리는 ffmpeg 기반으로 통일
- 처리 결과는 항상 /tmp 또는 output/ 디렉토리에 저장
- 한국어와 영어 모두 지원하도록 설계

### Don't
- 시스템 Python에 직접 설치하지 않는다
- API 키가 필요한 서비스를 기본 동작으로 넣지 않는다
- 사용자의 원본 파일을 수정하지 않는다
- 불필요한 추상화를 만들지 않는다 (3번 반복되기 전까지 추상화하지 않음)

## 기술 스택

| 영역 | 도구 | 이유 |
|------|------|------|
| STT | mlx-whisper | Apple Silicon 네이티브, 무료 |
| 오디오 처리 | ffmpeg | 범용, 로컬 |
| 비디오 다운로드 | yt-dlp | 오픈소스 표준 |
| Python 환경 | venv + pip | 단순, 표준 |
| 빌드/실행 | Python 3.10+ | 호환성 |
